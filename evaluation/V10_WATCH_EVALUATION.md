## Executive Summary

v10-watch is a dual-branch neural network trained on a hybrid dataset of real clinical signals and **realistic synthetic Apple Watch PPG**. The key innovation is using a physics-informed PPG generator that closely approximates actual Apple Watch optical sensor output — including wrist PPG morphology, green LED sensor noise, skin-tone dependent attenuation, gait artifacts, and contact quality variation.

**Performance highlights:**
- **Real ICU test (MIMIC/MMASH):** AUROC=1.000, F1=1.000, 110 signals
- **Realistic watch test:** AUROC=1.000, F1=0.994, 150 signals
- **130-synthetic comparison:** AUROC=0.987, F1=0.940 (best among v8/v9/v10)

## 1. Model Architecture

Dual-branch neural network (same as v7-v9):

```
PPG Branch:
  Input: (7500, 1) raw PPG @ 25 Hz
  → Conv1D(16, 7, relu) + BatchNorm + MaxPool(2)
  → Conv1D(32, 5, relu) + BatchNorm + MaxPool(2)
  → Conv1D(64, 3, relu) + BatchNorm + MaxPool(2)
  → BiLSTM(32)
  → GlobalMaxPool + GlobalAvgPool → Concatenate

Feature Branch:
  Input: (100,) hand-crafted features
  → Dense(32, relu) → Dropout(0.3)
  → Dense(32, relu) → Dropout(0.3)

Shared:
  Concatenate(PPG, Features) → Dense(32, relu)

Event Head:
  → Dense(16, relu) → Dense(1, sigmoid)
```

| Component | Output Shape | Parameters |
|-----------|-------------|------------|
| PPG branch (Conv + BiLSTM) | 128-dim | ~48K |
| Feature branch (MLP) | 32-dim | ~3.5K |
| Shared + event head | scalar | ~1.5K |
| **Total** | | **65,681** |

## 8. Why v10 Succeeds

### The Domain Gap Problem

Previous models (v8, v9) were trained on signals that don't resemble real Apple Watch output:

| Signal Type | Source | Morphology | Domain Match |
|-------------|--------|------------|--------------|
| MIMIC ICU | Finger PPG (arterial) | Sharp peaks, prominent dicrotic notch | Poor |
| MMASH wearable | Research device (wrist) | Pseudo-PPG from RR intervals | Poor |
| v8/v9 synthetic | 3-Gaussian pulses | Simple symmetric peaks | Poor |
| **v10 synthetic** | **RealisticWatchPPGGenerator** | **Broad peaks, weak notch, wrist physics** | **Good** |

### Key Innovations in v10

1. **Physics-informed PPG generator** — Models the actual optical physics of green LED PPG at the wrist:
   - Capillary bed diffusion broadens the systolic peak
   - Arterial wave reflection is dampened at the wrist (weak dicrotic notch)
   - Pulse transit time is longer (peak shifted to ~37% of cycle vs. 25-30% for finger)

2. **Skin-tone dependent optics** — Models melanin absorption at 530 nm (green LED):
   - Higher melanin → more absorption → lower AC amplitude
   - Higher melanin → higher DC offset → lower AC/DC ratio
   - Darker skin → lower SNR → more realistic noise

3. **Gait-synchronized artifacts** — Based on real accelerometer recordings:
   - Fundamental at gait frequency (1.0-2.5 Hz)
   - Harmonics at 2x, 3x (nonlinear arm swing)
   - Bursty envelope synchronized with foot strikes

4. **Apple Watch-specific noise** — Models the actual sensor:
   - Green LED photodetector: shot noise + thermal noise
   - 12-14 bit ADC quantization
   - SNR dependent on skin tone

## 9. Limitations

| Limitation | Impact | Mitigation |
|------------|--------|------------|
| **Synthetic training data** | May not capture all real-world variability | Domain adaptation techniques |
| **Small real dataset (59 patients)** | Limited generalization | Collect more real wrist PPG data |
| **Single-wrist PPG modality** | May miss non-PPG cardiac indicators | Add accelerometer, ECG if available |
| **Validation AUC plateaus at 0.958** | Slight overfitting to training distribution | More diverse training data |
| **Threshold sensitivity** | Watch test threshold (0.94) is high | May need calibration for deployment |
| **No prospective validation** | Performance on real patients unknown | Clinical trial required |
